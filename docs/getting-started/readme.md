# Getting started

A pi-lot-ui process serves the browser application and starts one `pi --mode rpc` child process for each active runner. The server requires Node.js 22.19 or newer and a usable pi CLI.

Follow [Installation](/getting-started/installation/) for a local checkout, then review [Configuration](/getting-started/configuration/) before exposing the server outside localhost.
