Oyster exposes pi's conversation workflow in a browser and adds operational tools around it.

- [Sessions](/user-guide/sessions/)
- [Credentials and OAuth](/user-guide/credentials/)
- [Files, routines, and hublots](/user-guide/extensions/)

Use the header and menu to change models, compact context, start sessions, manage credentials, and open sidebars. The composer sends with Enter, can steer an active response, and offers a Stop action while pi is running.
